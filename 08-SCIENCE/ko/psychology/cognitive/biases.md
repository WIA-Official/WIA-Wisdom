# Cognitive Biases | 인지 편향 (認知偏向)

> 사고의 체계적 오류: 우리의 정신적 지름길 이해하기

---

## Overview | 개요

인지 편향은 사람들이 정보를 처리하고 해석할 때 발생하는 무의식적이고 체계적인 사고의 오류입니다. 1970년대에 이스라엘 심리학자 아모스 트버스키와 다니엘 카너먼에 의해 처음 식별된 이러한 정신적 패턴은 종종 합리적 선택 이론에서 벗어나는 예측 가능한 방식으로 우리의 판단과 결정에 영향을 미칩니다.

편향은 휴리스틱—빠른 결정을 내리는 데 도움이 되지만 오류로 이어질 수 있는 정신적 지름길—에서 발생합니다.

---

## Two Systems of Thinking | 두 가지 사고 체계

> "Kahneman's model describes two parallel systems of thought that perform different functions."
| System | Characteristics | Function |
|--------|----------------|----------|
| **System 1** | Fast, automatic, intuitive | Quick judgments, patterns |
| **System 2** | Slow, deliberate, analytical | Complex reasoning, calculation |
### Key Insight:

| 체계 | 특징 | 기능 |
|-----|-----|------|
| **체계 1** | 빠름, 자동적, 직관적 | 빠른 판단, 패턴 |
| **체계 2** | 느림, 의도적, 분석적 | 복잡한 추론, 계산 |

---

## Major Heuristics | 주요 휴리스틱

### The Three Classic Heuristics (Tversky & Kahneman):
| Heuristic | Definition | Error |
|-----------|------------|-------|
| **Representativeness** | Judge by similarity to typical case | Ignore base rates |
| **Availability** | Judge by ease of recall | Overweight vivid events |
| **Anchoring** | Rely on initial reference point | Insufficient adjustment |

| 휴리스틱 | 정의 | 오류 |
|---------|-----|-----|
| **대표성** | 전형적 사례와의 유사성으로 판단 | 기저율 무시 |
| **가용성** | 회상 용이성으로 판단 | 생생한 사건 과대평가 |
| **정박** | 초기 참조점에 의존 | 불충분한 조정 |

## Common Cognitive Biases | 일반적인 인지 편향

| Bias | Description |
|------|-------------|
| **Confirmation Bias** | Seeking information that confirms beliefs |
| **Optimism Bias** | Overestimating positive outcomes |
| **Hindsight Bias** | "I knew it all along" after the fact |
| **Dunning-Kruger Effect** | Unskilled unaware of incompetence |
| **Loss Aversion** | Losses feel worse than equivalent gains |
| **Sunk Cost Fallacy** | Continuing due to past investment |
| **Planning Fallacy** | Underestimating time and costs |
| **Framing Effect** | Different decisions based on presentation |
| **Status Quo Bias** | Preference for current state |
| **In-group Bias** | Favoring one's own group |
> "Kahneman writes of a 'pervasive optimistic bias,' which 'may well be the most significant of the cognitive biases.'"

| 편향 | 설명 |
|-----|------|
| **확증 편향** | 믿음을 확인하는 정보 찾기 |
| **낙관주의 편향** | 긍정적 결과 과대평가 |
| **후견지명 편향** | 사후 "나는 알고 있었다" |
| **더닝-크루거 효과** | 미숙련자가 무능함을 인식 못함 |
| **손실 회피** | 손실이 동등한 이익보다 더 나쁘게 느껴짐 |
| **매몰 비용 오류** | 과거 투자 때문에 계속함 |
| **계획 오류** | 시간과 비용 과소평가 |

---

## Expert Vulnerability | 전문가 취약성

> "Even experts display biases when producing answers based on intuition, suggesting that our intuitions often derive from non-rational decision-making processes."

주요 발견:
- 전문성이 편향을 방지하지 못함
- 확신이 정확성을 나타내지 않음
- 편향 제거 의도적 연습이 도움됨
- 자각이 완화의 첫 단계

## Practical Applications | 실용적 적용

### Debiasing Strategies:
| Strategy | Application |
|----------|-------------|
| **Slow Down** | Engage System 2 |
| **Consider Alternatives** | Fight confirmation bias |
| **Use Base Rates** | Counter representativeness |
| **Pre-mortem Analysis** | Imagine failure before deciding |
| **Devil's Advocate** | Assign someone to argue against |
| **Checklists** | Reduce oversight errors |

| 전략 | 적용 |
|-----|------|
| **속도 줄이기** | 체계 2 관여 |
| **대안 고려** | 확증 편향과 싸움 |
| **기저율 사용** | 대표성에 대항 |
| **사전 검시 분석** | 결정 전 실패 상상 |
| **악마의 변호인** | 반대 주장할 사람 지정 |
