# Anekantavada | 다면적 진리 (अनेकान्तवाद)

> 다면성 교리: 현실은 다양한 측면을 가진다

---

## Overview | 개요

아네칸타바다(산스크리트어: "다면성" 또는 "비일원성")는 현실과 진리의 본질에 관한 자이나교의 근본 교리입니다. 이것은 현실이 복잡하고 다면적이며, 궁극적 진리는 여러 관점을 동시에 고려해야만 파악될 수 있다고 주장합니다. 이 교리는 "자이나교의 심장"으로 여겨집니다.

이 단어는 세 어근으로 구성됩니다: "안"(아닌), "에카"(하나), "안타"(끝, 측면)—함께 "한 측면이 아닌" 또는 "다측면"을 의미합니다.

---

## The Parable of the Blind Men | 장님과 코끼리 비유

> "Five blind men who have never seen an elephant each touch and describe different parts of it differently—as a branch, a rope, a wall, a pillar, or a fan."
### The Teaching:

> "코끼리를 본 적 없는 다섯 장님이 각각 다른 부분을 만지고 다르게 묘사합니다."

### 가르침:
- 한 사람이 코를 만짐: "나뭇가지 같아!"
- 한 사람이 꼬리를 만짐: "밧줄 같아!"
- 한 사람이 옆구리를 만짐: "벽 같아!"
- 한 사람이 다리를 만짐: "기둥 같아!"
- 한 사람이 귀를 만짐: "부채 같아!"

**모두 맞지만, 부분적으로만 맞습니다.**

## Related Concepts | 관련 개념

### 1. Nayavāda (Partial Viewpoints):
| Naya | Description |
|------|-------------|
| **Naigama** | Common sense viewpoint |
| **Sangraha** | Class viewpoint |
| **Vyavahāra** | Practical viewpoint |
| **Ṛjusūtra** | Present-moment viewpoint |
| **Śabda** | Verbal/etymological viewpoint |
| **Samabhirūḍha** | Conventional viewpoint |
| **Evambhūta** | Specific activity viewpoint |
### 2. Syādvāda (Conditional Predication):

### 1. 나야바다 (부분적 관점):
현실에 대한 일곱 가지 관점을 체계화합니다.

### 2. 샤드바다 (조건적 술어):
진리의 상대성을 표현하는 방법:
- "어떤 면에서는 그렇다"
- "어떤 면에서는 그렇지 않다"
- "어떤 면에서는 둘 다이다"
- "어떤 면에서는 표현할 수 없다"

## Philosophical Significance | 철학적 의의

> "Any real and determinate thing admits of contrary predications."
### Core Principles:
### Contrast with Other Views:
| Philosophy | Position |
|------------|----------|
| **Jainism** | All viewpoints contain partial truth |
| **Buddhism** | Middle way between extremes |
| **Advaita** | One ultimate reality |

### 핵심 원칙:
1. **현실은 복잡하다** - 어떤 단일 관점도 모든 측면을 포착하지 못함
2. **모든 관점은 진리를 포함** - 그러나 부분적 진리만
3. **겸손이 필수적** - 누구도 완전한 그림을 갖지 못함
4. **대화에는 개방성 필요** - 이해에는 다중 관점이 필요

## Modern Applications | 현대적 적용

> "Anekantavada has been interpreted to mean non-absolutism, 'intellectual Ahimsa,' religious pluralism, as well as a rejection of fanaticism."
### Contemporary Relevance:
| Application | Description |
|-------------|-------------|
| **Religious Dialogue** | Understanding different faiths |
| **Conflict Resolution** | Seeing validity in opposing views |
| **Scientific Inquiry** | Complementary theories (wave-particle) |
| **Democratic Discourse** | Valuing diverse perspectives |
| **Anti-Fundamentalism** | Rejecting absolutist claims |

| 적용 | 설명 |
