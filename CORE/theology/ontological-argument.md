# Ontological Argument | 존재론적 논증

> From concept to existence: God's existence from reason alone
> 개념에서 존재로: 이성만으로 신의 존재

---

## Overview | 개요

**English:**

The **Ontological Argument** is a philosophical argument for God's existence based solely on reason and the concept of God, without appeal to empirical evidence. First formulated by Anselm of Canterbury, it argues that if we properly understand what God is, we must conclude that God exists.

Unlike cosmological arguments (from the world) or teleological arguments (from design), the ontological argument is purely *a priori*.

> "God is that than which nothing greater can be conceived." — Anselm

**한국어:**

**존재론적 논증**은 경험적 증거에 호소하지 않고 이성과 신 개념에만 기초한 신 존재에 대한 철학적 논증입니다. 캔터베리의 안셀무스가 처음 공식화했으며, 신이 무엇인지 제대로 이해하면 신이 존재한다고 결론지어야 한다고 논증합니다.

우주론적 논증(세계에서)이나 목적론적 논증(설계에서)과 달리, 존재론적 논증은 순수하게 *선험적*입니다.

---

## Anselm's Version | 안셀무스 버전

**English:**

### The Argument:
1. God is defined as "that than which nothing greater can be conceived"
2. God exists in the understanding (we can think of God)
3. Existing in reality is greater than existing only in understanding
4. If God existed only in understanding, we could conceive something greater
5. But nothing greater than God can be conceived
6. Therefore, God must exist in reality

### The Key Move:
- Existence in reality > existence in mind only
- A God that only exists in thought is not the greatest conceivable
- Real existence is part of God's definition

**한국어:**

### 논증:
1. 신은 "그보다 더 큰 것이 생각될 수 없는 것"으로 정의
2. 신은 이해 속에 존재 (신을 생각할 수 있음)
3. 실재에 존재함이 이해에만 존재함보다 더 큼
4. 신이 이해에만 존재하면, 더 큰 것을 생각할 수 있음
5. 그러나 신보다 더 큰 것은 생각될 수 없음
6. 따라서, 신은 실재에 존재해야 함

### 핵심 움직임:
- 실재에 존재 > 마음에만 존재
- 생각에만 존재하는 신은 가장 큰 생각 가능한 것 아님
- 실제 존재가 신의 정의 일부

## Other Versions | 다른 버전

**English:**

### Descartes's Version:
- Existence is a perfection
- God is supremely perfect
- God must have all perfections
- Therefore God must exist

### Leibniz's Addition:
- First must prove God is possible
- If possible, God is necessary
- God's concept contains no contradiction
- Therefore God exists

### Plantinga's Modal Version:
- A maximally great being is possible
- If possible, exists in some possible world
- If exists in some world, exists in all worlds
- Therefore exists in actual world

**한국어:**

### 데카르트 버전:
- 존재는 완전함
- 신은 최고로 완전
- 신은 모든 완전함 가져야
- 따라서 신은 존재해야

### 라이프니츠 추가:
- 먼저 신이 가능함 증명해야
- 가능하면, 신은 필연적
- 신의 개념에 모순 없음
- 따라서 신 존재

### 플란팅가 양상 버전:
- 최대로 위대한 존재 가능
- 가능하면, 어떤 가능 세계에 존재
- 어떤 세계에 존재하면, 모든 세계에 존재
- 따라서 현실 세계에 존재

## Classic Objections | 고전적 반론

**English:**

### Gaunilo's "Lost Island":
- Conceive greatest possible island
- Same logic: must exist
- Absurd conclusion
- Argument proves too much

### Kant's Critique:
- "Existence is not a predicate"
- Existence adds nothing to concept
- 100 real thalers = 100 imaginary thalers
- Cannot define things into existence

### Hume's Point:
- No being's existence is necessary
- Can conceive non-existence of anything
- Existence always matters of fact
- Never matters of reason alone

**한국어:**

### 가우닐로의 "잃어버린 섬":
- 가능한 가장 큰 섬 생각
- 같은 논리: 존재해야
- 불합리한 결론
- 논증이 너무 많이 증명

### 칸트 비판:
- "존재는 술어가 아님"
- 존재가 개념에 아무것도 안 더함
- 100 실제 달러 = 100 상상 달러
- 정의로 존재하게 할 수 없음

### 흄의 요점:
- 어떤 존재의 실존도 필연적 아님
- 무엇이든 비존재 생각 가능
- 존재는 항상 사실 문제
- 이성만의 문제 절대 아님
